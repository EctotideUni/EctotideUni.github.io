Throughout creating this website, I've learnt quite a bit.
<!--more-->

I first created this website out of pure necessity, for a 4th year project at Abertay University, and so didn't care much for how it looked. However soon realised that it would be incredibly useful for showing what I've done, and decided to properly expand it, including adding this blog section.

I ended up using the Hacker theme, both because it gives stereotypical programmer vibes, and because the rounded style of the buttons makes the website look less flat, as a lot of modern website look.

To create this site how I wanted, especially in regards to presenting the projects I've worked on, I had to learn the [Liquid](https://shopify.github.io/liquid/) language. This wasn't particularly difficult, with Liquid being a rather simple language that adds onto existing HTML, CSS and Markdown, however, it's a useful experience regardless.
