This is a puzzle-platformer made in Unity for Abertay University's DES315 Professional Project module. I was a programmer and source control manager in a team of 8.
In particular I worked on the game's dialogue system, as well as the UI.
From this I learnt:
- The process of creating well-documented tools for others to implement
- Creating an accessible UI
- Communication skills for directing the team on which git branches to use
- Effectively resolving Git merge conflicts
