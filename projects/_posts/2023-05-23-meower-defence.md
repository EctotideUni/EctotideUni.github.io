A project I completed for my 1st year at Abertay Univertsity. It is a tower defence game built using the [SFML 2 framework](https://www.sfml-dev.org/). From this, I learnt how to create a game without a game engine, as well as how to scope game development projects for a deadline. I plan on returning to this at some point in the future and updating the design and functionality of the game.
